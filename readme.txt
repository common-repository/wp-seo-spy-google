== Description ==

En pocas palabras, Seo Spy comprueba su ranking diario de Google directamente en el interior de Wordpress.
 
Más precisamente, es posible introducir tres direcciones de Internet y tres palabras clave. posteriormente, Seo Spy comprobará cuáles de estos sitios están listados en Google Top 100. El plugin se instala dentro en sólo unos pocos segundos y ofrece una gran cantidad de información útil.
 
El plugin funciona de la siguiente manera: Seo Spy se conecta al servidor de Google, como si fuera un usuario normal y hace solicitudes de 3 sitios (como máximo). A continuación, los datos se guardarán y se incorporan. Esto explica la limitación de la cantidad de sitios y palabras clave. si Seo Spy pediría muchos más sitios, podría haber consecuencias negativas. (Esto es en Google el bloqueo de la IP solicitante.)

== Installation ==

1. Sube completamente la carpeta de `seo-spy` en el directorio `/wp-content/plugins/`
2. No le cambies el nombre!
3. Activa el plugin en la página de 'Plugins' en tu menú de WordPress
4. Inserta las palabras clave y URLs en las opciones de Seo Spy. 
5. Clic en `Actualizar`

== Frequently Asked Questions ==

= ¿Por qué puedo definir tan pocas palabras? =
Esta limitación está puesta para su protección. Si hay demasiadas solicitudes de búsqueda enviadas a Google por una misma IP, puede ocurrir, que Google bloquee la IP por un tiempo. Por lo tanto, más vale prevenir que curar.

= ¿Cuándo se actualizan los datos? =
La tarea de actualización manual se recomienda ejecutar todos los días a medianoche.

-------------------------------------------

= Why I can define as few words? =
This limitation is set for protection. If too many search requests sent to Google by the same IP, it may be that Google block the IP for a while. Therefore, prevention is better than cure.

= When is the data updated? =
The task of manually updating is recommended to run every day at midnight.
